# And gate
## Made with NPN transistors
### Breadboard version

**Some mistakes that I have made and learned from:**

- The LED has to come **BEFORE** the transistors, otherwise transistor before
the LED will act really like a diode and current will flow through the LED.
I don't know how mine did not burn.

- Always put a resistor before the transistor base, it is pretty obvious why.

- See the PDF for more details!
